Wir erstellen eine Anwendung, in der Rezepte gespeichert, verwaltet und gesucht werden können nach Rezeptname, Kategorien und Zutaten.

Link zum Frontend-Repository: https://github.com/lisatran03/prog3-frontend

Uses cases:
- Rezept erstellen
- alle Rezepte anzeigen
- Rezept bearbeiten
- Rezept löschen
- Nach Rezeptname suchen
- nach Kategorie filtern
- zwischen Dark/Light Mode wechseln (Frontend-Use Case)
